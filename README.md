# 设计师人格基因测试

> 又一个告诉你"你是谁"的测试。但这次，至少荒谬得更有品味一点。

---

## 为什么做这个

这个时代有一种奇特的病症：每个人都很焦虑，每个人都很迷茫，但每个人都迫不及待地想给自己贴一个标签。

MBTI 告诉你你是 INFJ 还是 ENTJ，星座告诉你水逆该注意什么，九型人格告诉你属于第几号。我们一边怀疑一切，一边疯狂地用四个字母定义自己的灵魂——仿佛知道了自己的"类型"，就能在这个越来越看不懂的世界里锚定些什么。

我对自己其实挺清楚的。但我看到身边越来越多人——特别是设计师——陷入一种诡异的循环：在社交媒体上表演"独特的自我"，同时拼命寻找某种归类来确认这种独特性。**我们到底是在寻找自我，还是在寻找一个标签来替代思考？**

所以随手做了这个东西。带着一些荒谬，一些虚无主义，也许还有一点真诚。

## 一个荒谬的前提

这个测试的前提本身就很荒谬：用32道选择题和3道问答，就能把你和110位横跨两千五百年的设计大师对号入座？

**当然不能。**

但 MBTI 用 93 道题就能把 80 亿人分成 16 种类型，大家不也玩得很开心吗？

选择"基因"这个隐喻而不是"类型"，是故意的。基因意味着混合和不确定性——你可能有 47% 的安藤忠雄、31% 的原研哉、22% 的扎哈·哈迪德。不是一个干脆利落的标签，而是一堆比例、一种模糊的倾向。这更接近真实，也更接近虚无：它告诉你一些什么，但你仔细想想，它又什么都没说。

110 位设计大师从春秋鲁班到当代比亚克·英格斯，覆盖建筑、平面、工业、室内、景观、理论/交互、时尚七个领域。这不是一份排行榜，而是一张坐标网——足够大，大到每种人格倾向都能在里面找到映射。也足够荒诞，荒诞到把给古人设计陵墓的李诫和给外星人设计概念建筑的扎哈放在同一张量表上。

## 焦虑、效率和正在消失的东西

AI 能在 3 秒内出一张海报。于是一个问题变得尖锐：**当"做设计"这件事可以被自动化的时候，设计师还剩下什么？**

答案大概不是"更快地出方案"。也不是"更好地使用 AI 工具"。

路易·康在萨尔克研究所追问"光想要什么"的时候，没有人会觉得这个问题有"效率"。王澍捡起一块废弃的青砖，脑子里浮现的是文人山水而不是 KPI。川久保玲每一季都在解构"美"这个概念本身——这不是一种"高效的工作方式"，这是一种存在方式。

但我们这个时代不太允许这种存在方式了。甲方说"参考一下竞品"，我们说"好的"。Dribbble 的点赞数变成了审美标准。"用户体验"被简化为"转化率"。设计正在从一种对世界的回应，变成一种可以被量化、被提效、被 AI 替代的服务。

> 也许焦虑的根源不是"我会不会被替代"，而是"我本来就没想清楚自己在做什么"。

这个测试里有一个维度叫"传统 vs 创新"。鲁班的榫卯和扎哈的参数化曲线之间隔了两千五百年，但他们面对的是同一个问题：如何在既有的世界里，创造一个尚不存在的东西？这个问题没有标准答案。测试也不会给你标准答案。它只是让你在选择的过程中，可能——只是可能——想一想。

## 它做了什么

32 道选择题，前两道故意设计得比较荒谬。五个人格维度：秩序与混沌、理性与感性、传统与创新、个体与集体、物质与精神。3 道开放题。系统从 110 位设计大师中匹配三位，生成你的"设计基因图谱"。瑞士国际主义风格界面——网格、无衬线、克制。AI 可用时给深度解析，不可用时本地算法兜底。

认真做的。但别太认真地对待结果。
